# Guía de Estilos UI (El Tema Matemático)

La aplicación **ProfeApp** tiene impuesta una identidad visual autotitulada **"Matemático"**. Todo diseño UI de los agentes que operen bajo esta estipulación deberá respetar la guía semántica.

## Colores

- **`theme.colors.primary` (#1d4ed8 - Azul Matemático/Royal):** Asociado al cerebro lógico, tinta de bolígrafo o pizarras de acrílico. Usarlo en encabezados, tabs de la universidad, y acciones primarias.
- **`theme.colors.secondary` (#15803d - Verde Pizarra):** Asociado a elementos orgánicos y clientes y usuarios. Utilízalo en interacciones particulares o acentos secundarios.
- **`theme.colors.background` (#f8fafc - Slate):** Todo el background general para reducir fatiga visual del blanco puro.
- **`theme.colors.surfaceVariant` (#e2e8f0):** Excelente para bordes o divisiones de lista.

## Formas de Vida
- Emplear siempre los **MaterialCommunityIcons** provistos por expo (Ej: `calculator`, `function-variant`, `account-supervisor`, `book-open-outline`).
- Las tarjetas (`components/LessonCard.tsx`) se recomiendan usar con bordes izquierdos coloreados y badges para indicar visualmente rápida lectura.

Siempre priorizar: **Simplicidad ante todo.** La interfaz es de uso rápido y consultivo. Evita modales sobrecargados y prefiere vistas simples y verticales limpias.
